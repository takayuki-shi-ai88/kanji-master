# ✏️ 漢字マスター 〜漢検チャレンジ〜

タッチペン(Apple Pencil)で漢字を書いて覚える、iPad 向けの漢字練習 Web アプリ(PWA)です。

## 特徴

- **漢検 10級〜1級**(準2級・準1級を含む全12レベル)から級を選んで練習
- **Apple Pencil 対応**の手書きキャンバス(筆圧対応・「ペンだけで書く」モードで手のひら誤反応を防止)
- **段階式ヒント**:ヒントボタンを押すたびに答えに近づく4段階
  1. 画数 + 部首・形のヒント
  2. 意味・覚え方のヒント
  3. マスにうすいお手本を表示(なぞり書き)
  4. 正しい書き順のアニメーション
- **書き順判定**:文字が合っていれば正解。ただし書き順が違う場合はどこが違うかを指摘し、番号つきアニメーションで正しい書き順を表示
- **きれいに書くコツ**:とめ・はね・はらい、横画の右上がりなど、書いた字に合わせたアドバイスを表示
- 級ごとのクリア状況を保存(端末内に記録)

書き順データは [KanjiVG](https://kanjivg.tagaini.net/)(CC BY-SA 3.0)を CDN から取得します。
一度読み込んだ漢字は端末にキャッシュされ、オフラインでも使えます。

## iPad で使うまでの手順

このアプリは Web アプリなので、どこかのサーバーに置いて iPad の Safari で開きます。
いちばん簡単なのは **GitHub Pages** で公開する方法です。

### 方法A: GitHub Pages で公開する(おすすめ)

1. このフォルダを GitHub のリポジトリに push する
2. リポジトリの **Settings → Pages** で
   - Source: `Deploy from a branch`
   - Branch: `master` / `(root)` を選んで Save
3. 数分後に表示される URL(`https://<ユーザー名>.github.io/<リポジトリ名>/`)を iPad の Safari で開く
4. 共有ボタン → **「ホーム画面に追加」** を押すと、アプリのように全画面で使えます

### 方法B: PC でローカルサーバーを立てて同じ Wi-Fi の iPad から開く

```powershell
# このフォルダで(Python がある場合)
python -m http.server 8000
```

PC の IP アドレス(`ipconfig` で確認)を使って、iPad の Safari で
`http://<PCのIPアドレス>:8000` を開きます。
※ Service Worker(オフライン機能)は HTTPS か localhost でのみ有効です。

## 使い方

1. 級を選ぶ
2. 問題文の【読み】に当てはまる漢字をマスに書く
3. わからないときは「💡 ヒント」を押す(押すほど答えに近づく)
4. 「✅ こたえあわせ」で判定
   - ◎ 文字も書き順も正解
   - ○ 文字は正解、書き順に注意(正しい書き順をアニメで表示)
   - ✗ 画数や形が違うときはやり直し
5. 「つぎの問題 ▶」で次へ

## 収録漢字について

各級 8 字ずつ、その級の代表的な漢字を収録したサンプル構成です。
`index.html` 内の `LEVELS` 配列に
`{ k:"漢字", yomi:"よみ", q:"問題文(★が漢字の位置)", h1:"ヒント1", h2:"ヒント2" }`
の形式で追加すれば、自由に問題を増やせます(書き順データは自動取得されます)。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | アプリ本体(画面・判定ロジック・問題データすべて) |
| `manifest.json` | PWA マニフェスト(ホーム画面追加用) |
| `sw.js` | Service Worker(オフラインキャッシュ) |
| `icon.svg` | アプリアイコン |
